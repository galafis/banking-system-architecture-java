Banking System Architecture (Java)

Este é um projeto de demonstração para uma arquitetura de sistema bancário robusta e escalável desenvolvida em Java com Spring Boot.

Diagrama de Arquitetura

Funcionalidades Principais

  • Gerenciamento de Contas
  • Operações Financeiras (Depósito, Saque, Transferência)
  • API RESTful
  • Persistência de Dados com JPA e H2 Database
  • Testes Unitários e de Integração

Como Usar a API

A API pode ser acessada em http://localhost:8080/api/accounts após iniciar a aplicação. Abaixo estão alguns exemplos de requisições:

Criar uma nova conta

curl -X POST http://localhost:8080/api/accounts \
-H "Content-Type: application/json" \
-d '{"accountNumber": "1001", "accountHolderName": "Gabriel Lafis", "balance": 1500.00}'

Depositar em uma conta

curl -X PUT http://localhost:8080/api/accounts/1001/deposit \
-H "Content-Type: application/json" \
-d '{"amount": 200.00}'

Sacar de uma conta

curl -X PUT http://localhost:8080/api/accounts/1001/withdraw \
-H "Content-Type: application/json" \
-d '{"amount": 150.00}'

Transferir entre contas

curl -X PUT http://localhost:8080/api/accounts/transfer \
-H "Content-Type: application/json" \
-d '{"fromAccountNumber": "1001", "toAccountNumber": "1002", "amount": 50.00}'

Para mais detalhes, consulte o README.md no repositório.
